# Super Star Trek: 24th Century Edition

A modern web-based port of the classic 1970s BASIC computer game *Super Star Trek*. This version features an LCARS-inspired interface and sound effects.

## Features

- **Classic Gameplay**: Command the USS Enterprise to destroy Klingon warships across the galaxy.
- **LCARS Interface**: Authentic Star Trek-style dashboard for navigation, tactical, and operations.
- **Sound Effects**: Immersive audio for warp, phasers, torpedoes, and alerts.
- **Save the Galaxy**: Track your stardate and fulfill your mission before Federation Headquarters is attacked.

## Getting Started

To play the game, simply open `startrek.html` in any modern web browser.

## Controls

- **Navigation (`nav`)**: Control the ship's course and warp factor.
- **Short Range Scan (`srs`)**: View the current quadrant and ship status.
- **Long Range Scan (`lrs`)**: Scan surrounding quadrants.
- **Phasers (`pha`)**: Fire energy weapons at Klingon ships.
- **Torpedoes (`tor`)**: Launch photon torpedoes.
- **Shields (`she`)**: Manage shield energy.
- **Damage Control (`dam`)**: Check the status of ship systems.
- **Computer (`com`)**: Access the library computer for tactical information.
